# Jarnisha · Sourdough Field Manual

A dark, terminal-styled "field manual" for keeping a sourdough starter alive and baking a loaf from it. Covers the core concept of a starter, the 1:1:1 feeding ratio, counter vs. fridge maintenance, the float test for readiness, a classic Dutch-oven boule recipe, plus troubleshooting and golden rules.

## What's in here

The whole guide is a single self-contained file: `jarnisha-field-manual.html` — no build step, no dependencies.

## Run locally

Just open `jarnisha-field-manual.html` in any browser. That's it.
