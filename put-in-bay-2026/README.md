# Put-in-Bay · May 23, 2026

A single-page trip planner for the group’s Memorial Day weekend voyage to Put-in-Bay.

## What’s in here

- `index.html` — the entire app. Single file, no build step, no dependencies beyond Google Fonts.

## Run locally

Just open `index.html` in any browser. That’s it.

## Deploy

**GitHub Pages (recommended for sharing with the group):**

1. Push this repo to GitHub.
1. Go to **Settings → Pages**.
1. Set source to `main` branch, root folder.
1. Wait ~1 minute. Your link will be `https://<your-username>.github.io/<repo-name>/`.
1. Share that link.

## Features

- Live countdown to 1:00 PM May 23.
- Hour-by-hour itinerary with hard deadlines marked.
- Place roster (bars, food, backups).
- Logistics for ferry / bus / parking with tap-to-call.
- Persistent bring-list (checks save to `localStorage`).
- Mobile-first — designed to be read on phones day-of.

## The hard rules

- **Last bus from The Depot: 7:40 PM**
- **Last ferry from Lime Kiln: 8:00 PM**
- **Cash only on the bus ($3/person each way)**
- **Miss either = stranded overnight**
